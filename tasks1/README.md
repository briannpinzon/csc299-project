\# Distrait 



Distrait is a simple program that helps you track important tasks in a JSON file 



\## Commands 



\# \*\*add\*\*



Adds a new task to a JSON file

i.e. \*python add "desired task here"



\# List



Lists all tasks

i.e. \*python list\*



\# Search



Searches for a task

i.e. \*python search keyword\*
